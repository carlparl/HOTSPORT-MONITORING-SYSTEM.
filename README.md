# HotSpot Pro - Complete Hotspot Management System

## Features
- Admin dashboard with real‑time stats, voucher management, user monitoring
- Captive portal with voucher login, package purchase, free trial
- Node.js backend with MongoDB for data persistence
- MikroTik integration scripts
- All prices in UGX

## Installation
1. Install Node.js and MongoDB.
2. Clone or copy all files into a folder named `hotspot-pro`.
3. In the `api` folder, run `npm install`.
4. Create a `.env` file in `api` with:








# *for me/*
# PORT=3000
## MONGODB_URI=mongodb://localhost:27017/hotspot_pro ##
