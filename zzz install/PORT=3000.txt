## Usage
- Admin: create vouchers, view active users, check payments.
- Customers: enter voucher code or buy a package to get WiFi access.

## Customisation
- Modify `portal-styles.css` to change colours.
- Adjust prices and packages in `portal-scripts.js` and backend logic.
- Replace simulated API calls with real integration to MikroTik.
